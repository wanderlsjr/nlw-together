# NLW - Letmeask

Projeto criado durante a 6ª edição do NLW da Rocketseat.

## Sobre a aplicação

Letmeask é uma aplicação que auxilia criadores de conteúdo a receber e responder perguntas de seus usuários

### Tecnologia utilizada

React\
Typescript\
Firebase
